 - SCSS
 - ES6 + Babel
 - Webpack 3

### How to run
- **yarn** or **npm** install first after cloning the library.
- For the development setup , run **yarn dev/npm run dev**.
- For the Production setup, run **yarn prod/npm run dev**.
